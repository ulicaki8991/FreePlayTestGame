FR — Cartoon Shader - URP



____________________________________________________________________________________________________




Version compatible : Unity 6000.0.33f1  



_____ Installation _____

1. Créer un matériau et assigner le Cartoon Shader.
2. Appliquer le matériau sur un modèle 3D.
3. Le shader utilise la lumière principale de la scène pour générer les ombres et les highlights.  



_____ Paramètres disponibles _____

Main parameters :  
- Base color  
- Shadows color  
- Shadows contrast  
- Shadows size  

Outline :  
- Outline size  
- Outline color  

Highlight :  
- Highlight size  
- Glow size  
- Highlight color  
- Highlight intensity  



_____ Modification des gradients _____

Les gradients définissant les ombres et les highlights ne sont pas modifiables dans l'éditeur de matériaux. Ils sont stockés sous forme d'images dans le Shader Graph (format Photoshop par défaut).

Pour les modifier :
1. Ouvrir le fichier Photoshop correspondant.
2. Modifier le gradient selon les besoins.
3. Sauvegarder et remplacer l'image utilisée dans le Shader Graph.
